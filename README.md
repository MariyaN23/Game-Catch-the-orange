# 🍊 Game: Catch The Orange 🍊

## Application Description

In the game, capybaras try to catch as many 🍊 as possible

## Functionality
- 2 players in the game
- One player controls with ⬅️⬆️⬇️➡️
- Another player controls with 🆆🅰🆂🅳
- You can set the number of points to win
- And you need to catch as many 🍊 as possible!

## Stylization
The app is styled with CSS

## Tools
- JavaScript
- OOP principles
- Event Emitter Pattern
- Dependency injection to play the game locally (by default) or through a ws
